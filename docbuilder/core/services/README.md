# Core Services Module

Módulo que implementa a camada de aplicação/serviços (Casos de Uso) do GoTryx Documentation Builder.

## Componentes
* `dtos.py`: Estrutura de dados serializáveis de comunicação externa.
* `project_services.py`: Casos de uso de gerenciamento e persistência de projetos.
* `build_services.py`: Casos de uso de validação e compilação do pipeline.
