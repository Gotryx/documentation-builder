# UI Module

Interface gráfica Desktop baseada em PySide6 (Qt) seguindo a estética premium e as diretrizes visuais da GoTryx.

## Componentes
* `main_window.py`: Gerencia a janela principal, conexões de sinais de eventos e controles da árvore.
* `styles.py`: Contém a folha de estilo unificada QSS (tema escuro elegante).
